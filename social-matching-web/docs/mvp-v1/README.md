# Social Matching MVP v1

זוהי ספריית האפיון הראשית הראשונה של הפלטפורמה החדשה.

המטרה של התיקייה הזו היא להגדיר את המערכת **כמוצר עצמאי**, בלי framing של migration, pivot או השוואה למערכת קודמת.

## מטרת הסט

המסמכים כאן מגדירים:
- מהו המוצר
- מה נכנס ל-MVP
- מי סוגי המשתמשים
- איך כל workflow עובד
- מהן הדרישות הפונקציונליות
- מהו הבייסליין הטכני
- אילו user stories מגדירות את הליבה

## עקרון מנחה

המוצר מוגדר כאן כפלטפורמה ל:
- גילוי מפגשים קטנים
- הגשת מועמדות
- התאמה ואישור
- תשלום אחרי אישור
- ניהול קבוצה ומפגש
- פידבק ואמון
