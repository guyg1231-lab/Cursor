# Circles Copy Glossary (HE -> EN)

## Purpose
Canonical wording map for consistent Hebrew and English product copy.

## Core Product Terms
- מעגל -> circle
- מעגלים -> circles
- מפגש -> gathering
- מפגשים -> gatherings
- אירוע -> event
- אירועים -> events
- חוויה משותפת -> shared experience
- חיבור אמיתי -> real connection
- קבוצה קטנה -> small group

## Participant Journey Terms
- פרופיל -> profile
- שאלון -> questionnaire
- התאמה -> matching fit
- עדכון -> update
- שלב הבא -> next step
- אזור אישי -> personal area

## Admin And Host Terms
- אדמין -> admin
- מארח/ת -> host
- בקשת אירוע -> event request
- הרשמה -> application
- סטטוס -> status

## CTA Standards
- לצפייה במפגשים -> view gatherings
- להשלמת פרופיל -> complete profile
- למציאת מעגל -> find your circle
- להצטרפות לפיילוט -> join the pilot

## Do Not Swap Without Decision
- מעגל <-> קהילה (do not interchange casually)
- מפגש <-> אירוע (choose by context and stay consistent in the section)
- התאמה <-> סינון (avoid "screening" tone unless admin context)

## Forbidden Or Discouraged Terms
- רנדום
- רנדומליות
- find your tribe
- life-changing
- best-in-class
- no drama
