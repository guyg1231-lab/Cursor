This app is the new forked-build implementation surface for the social matching MVP pivot.

## Important Rule

This app is built under the **Forked Build / No-Touch Legacy** model:

- the legacy app remains intact
- reused logic/components are copied into this app
- adaptations happen here
- legacy code is reference/donor material, not the primary construction site
